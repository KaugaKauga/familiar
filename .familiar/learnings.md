- The `Pipeline` struct is serialized/deserialized with serde for both JSON (state.json migration) and SQLite persistence — any new field needs a default value or the migration path will fail for existing databases.
- Git worktree `.git` is a file (not a directory) pointing to the main repo's `.git/worktrees/<name>/` — lock files like `index.lock` live there, not at `worktree/.git/index.lock`. Use `git rev-parse --git-dir` to resolve it.
- The `cleanup_run()` method on Pipeline is `#[allow(dead_code)]` — actual cleanup happens in the orchestrator loop in `main.rs` via a spawned tokio task, not through this method.
- `db.rs` migrations use `PRAGMA table_info` to detect missing columns and `ALTER TABLE` to add them — new columns must have a `DEFAULT` clause to work with existing rows.
- When adding a new field to `Pipeline`, you must update 5 places: the struct definition, `Pipeline::new()`, `get_all_active_pipelines()` SELECT, `upsert_pipeline()` INSERT, and `migrate_from_state_json()` INSERT.
- The `do_fix()` method has two modes: pre-submit (no PR yet, returns to Verify) and post-submit (PR exists, commits/pushes and returns to Watch). The `pr_number` field determines which path is taken.
- The repo-root shim (`./familiar`) just `exec`s `cargo run --manifest-path daemon/Cargo.toml` — so renaming the Cargo package name in `daemon/Cargo.toml` is enough to rename the binary; no build-script or `[[bin]]` override is needed.
- Ratatui ships a `backend::TestBackend` that renders widgets into an in-memory `Buffer` — iterate `buf.content.iter().map(|c| c.symbol())` to reconstruct a plain string and assert on it. Useful for unit-testing TUI header/label changes without a real terminal.
- Brand/user-facing strings live in a surprising number of places: banner art, TUI header, `#[command(name = …)]` in clap, default `--label` value, log filenames, DB filenames, env-filter target (`"X=info"`), branch-name format, commit/PR title prefixes, git `user.name`/`user.email`, `@X` mention trigger, and `.X/learnings.md` path. A rename isn't complete until all 10+ are updated.
